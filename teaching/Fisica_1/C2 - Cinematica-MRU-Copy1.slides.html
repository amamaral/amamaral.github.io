Unidades de medida e conversão de unidades¶

Sistema internacional de unidades (ou MKS)¶

Conjunto de padres de medida aceito internacionalmente. Algumas grandezas fundamentais do SI:

  • Kg (massa)
  • m (distncia)
  • s (tempo)
  • Outras grandezas em mecnica so obtidas a partir das fundamentais acima.
  • Ex.: N = kg $\cdot$ m / s$^2$

Usage¶

  • Create a Jupyter Notebook (example.ipynb) in a given folder (e.g.: slides/)
  • Within slides/, add reveal.js
    • git clone https://github.com/hakimel/reveal.js.git, or
    • download and unzip reveal.js.
  • Copy slides_template.tpl and basic_mod.tpl to slides/
  • Run jupyter nbconvert --to slides --template slides_template.tpl example.ipynb --reveal-prefix=reveal.js
    • --reveal-prefix={folder} indicates where reveal.js folder is located

import matplotlib.path as mpath
import matplotlib.patches as mpatches
import matplotlib.pyplot as plt


fig, ax = plt.subplots()

Path = mpath.Path
path_data = [
    (Path.MOVETO, (1.58, -2.57)),
    (Path.CURVE4, (0.35, -1.1)),
    (Path.CURVE4, (-1.75, 2.0)),
    (Path.CURVE4, (0.375, 2.0)),
    (Path.LINETO, (0.85, 1.15)),
    (Path.CURVE4, (2.2, 3.2)),
    (Path.CURVE4, (3, 0.05)),
    (Path.CURVE4, (2.0, -0.5)),
    (Path.CLOSEPOLY, (1.58, -2.57)),
    ]
codes, verts = zip(*path_data)
path = mpath.Path(verts, codes)
patch = mpatches.PathPatch(path, facecolor='r', alpha=0.5)
ax.add_patch(patch)

# plot control points and connecting lines
x, y = zip(*path.vertices)
line, = ax.plot(x, y, 'go-')

ax.grid()
ax.axis('equal')
plt.show()
