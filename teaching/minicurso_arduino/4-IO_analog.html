O conversor analógico digital (ADC)

  • No arduino existe um periférico que consegue converter um sinal analógico em digital, denominado ADC (Analog Digital Converter) no datasheet.
  • Este conversor mede uma tensão entre 0 e 5V com uma resolução de 10 bits. Ou seja, o intervalo 0-5V é dividido em 210 = 1024 intervalos, e o resultado da medição de tensão é uma aproximação
  • Quanto maior a resolução, menor o tamanho dos intervalos e menor o erro
  • Em geral, ADC's de alta resolução operam a baixas velocidades. O ADC do arduino possui uma precisão razoável, e permite realizar até 10.000 medidas por segundo (10Ksps).
  • Tipicamente um osciloscópio digital possui uma resolução de 8-10 bits.

Sinal analógico de um potenciômetro

  • O potenciômetro funciona como um divisor de tensão, onde podemos variar a resistência de ambos os resistores.
  • Variando a posição do botão, podemos variar a tensão do pino do meio suavemente entre as tensões aplicadas nos pinos externos.
  • É muito usado para calibrar partes de circuitos.
  • Usaremos um potenciômetro para medir a posição do botão.
  • Nota: um potenciômetro é utilizado para determinar a posição em servo-motores, como o da figura...

Detecção de sinais analógicos usando o arduino

  • Conectaremos o potenciômetro conforme abaixo:

  • Abram o exemplo em Arquivo/Exemplos/01. Basics/AnalogReadSerial.

    void setup() {
  Serial.begin(9600); //Inicializa a comunicacao
}
void loop() {
  // Le sinal analogico no pino 0:
  int sensorValue = analogRead(A0);
  Serial.println(sensorValue);
}
  • int sensorValue = analogRead(A0); lê o sinal na porta analógica A0 e armazena na variável sensorValue
  • Vejam o que acontece com o valor exibido no monitor serial, que corresponde ao valor de sensorValue, quando a posição do potenciômetro muda.

Exercício

  • O sinal mostrado no monitor serial corresponde a um valor inteiro associado à medida de tensão. Sabendo que a resolução da medida é 10 bits (0-1023), e que o valor máximo corresponde a 5V, convertam sensorValue para tensão.
  • Nota: Em geral, quando utilizamos algum sensor, este sensor converte uma grandeza física em tensão. (ex.: OC/V), enquanto o arduino mede tensões. Desta forma, para usar um sensor precisamos converter o sinal a ser medido para tensão para então saber qual foi o valor da grandeza medida.

Produzindo um sinal analógico

  • Na etapa anterior medimos o sinal analógico proveniente de um potenciômetro. O arduino pode ser usado para perceber o que acontece ao seu redor.
  • Queremos também que o arduino possa ser usado para controlar o ambiente (operando um motor, acendendo uma lâmpada/led, aquecendo ou esfriando um dispositivo...)
  • Se a medida de uma tensão é realizada por um ADC (conversor analógico digital), a produção de uma tensão pelo arduino é realizada por um DAC (Digital Analog Converter).

Modulação de largura de pulso (PWM)

  • Existem vários tipos de DAC. Em especial, o do arduino opera usando um sinal tipo PWM (Pulse-Width Modulation).
  • O arduino envia um sinal digital com um tempo variável. A forma que o PWM do arduino opera não é exatamente conforme a figura acima, pois não existem pulsos negativos.
  • A média da tensão no tempo é fornece o sinal analógico de saída. O valor médio pode ser controlado facilmente.
  • a função analogWrite implementada no arduino permite gerar tensões analógicas com uma resolução de 8 bits (28 = 256 valores de tensão entre 0 e 5V).

Exercício: Controle suave da intensidade de um LED

  • Em nossos exercícios anteriores o LED sempre esteve ligado no seu máximo, ou então desligado. Usando PWM podemos obter uma intensidade mais agradável.
  • Será necessário um LED, um resistor de 220 Ohms, fios e uma protoboard.

  • Abram o exemplo Exemplos\01.Basics\Fade

    int led = 9;
int brightness = 0;
int fadeAmount = 5;

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  analogWrite(led, brightness);
  brightness = brightness + fadeAmount;

  if (brightness <= 0 || brightness >= 255) {
    fadeAmount = -fadeAmount;
  }
  delay(30);
}

  • Vejam o que acontece com a intensidade do LED. a função analogWrite varia o brilho do LED a cada iteração do loop. Após modificar o brilho, varia o valor. brightness é mantido sempre entre 0 e 255, já que a resolução do analogWrite é de 8 bits.

  • Façam com que o arduino meça a tensão de um potenciômetro controlar o brilho do LED. Obs.: neste caso a intensidade não deve variar no tempo automaticamente...
