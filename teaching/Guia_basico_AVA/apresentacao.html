Introdução

  • Proposta de um curso breve, focando nos aspectos principais
  • Como configurar um curso?
  • Como elaborar atividades valendo nota? (Exames Escolares)
  • Que tipo de questões podemos utilizar? Como inserir na plataforma?
  • Como obter as notas do AVA e inserir no SIG@?
  • Série de vídeos disponibilizada para todos.

Ambiente virtual de aprendizagem (AVA)

  • Ambiente virtual institucional (Moodle)
  • Possui integração direta com o SIG@
  • Testes com aplicação e correção automatizada
  • Pode-se gerar questões dinamicamente

AVA permite

  • Colaboração intertemporal: testes e avaliações podem se tornar um banco de dados que será enriquecido ao longo do tempo.

Acompanhamento detalhado

Configurações básicas do curso

  • Considerarei um curso com 2 exercícios escolares e 1 lista de exercícios
  • Página da disciplina contém algumas informações básicas, como link para o informe inicial
  • Visão geral de um curso já em operação (Backup de Física Geral 1)
    • Mensagem de boas vindas
    • Página inicial
    • Agendando atividades (EE’s, listas de exercício)
  • Uma única configuração para todas as turmas de uma dada disciplinas

Visão geral de um curso já configurado

  • Página inicial
  • Organização das informações em abas
  • Upload de arquivos, links, atividades em geral, informações para a equipe
  • Informações/atividades podem ser ocultadas dos estudantes
  • É possível ver como estudantes visualizam a página “mudando o papel”

Configurações básicas

  • Mensagem de boas vindas para não cadastrados
  • Informações iniciais
  • Gerando 2 EE’s e 1 lista de exercício.
    • 2x questionários
    • 1x tarefa

Gerando questões para avaliações

  • Banco de questões e categorias
  • Tipos de questões principais
    • Dissertação
    • Múltipla escolha
    • Múltiplo verdadeiro falso
    • Calculado
    • Respostas embutidas (cloze)

Inserindo questões na avaliação

Correção de atividades

  • Questões dissertativas devem ser corrigidas manualmente
  • Todas as outras são corrigidas automaticamente
    • Caso haja algum problema na correção, é possível mudar a questão e reavaliar todas as notas atribuídas

Extraindo as notas do AVA

  • O AVA permite exportar as notas para o Excel, numa lista idêntica à do SIG@.
