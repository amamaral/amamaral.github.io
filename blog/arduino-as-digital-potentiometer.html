Arduino as digital potentiometer

Anderson M. Amaral

27/04/2018

Ativar/desativar resistores por meio das saídas digitais, alterando o estado de impedância + o estado da saída. Seria equivalente a uma escada R-2R, porém que pode ser usada para também para controlar o ganho de amplificadores externos, e outros ajustes. Lembra também o uso do pino digital no exemplo de carga/descarga do capacitor.

Ideia extremamente útil para produzir um conversor de sinal PWM -> onda triangular -> onda quase-senoidal, visto que será necessário ajustar o integrador e/ou a amplitude da onda na saída.

# The code below states the available PWM frequencies given
# a set of pre-scalers
%pylab inline
OCRA = arange(0, 256)
prescaler_values = [1, 8, 64, 256, 1024]
clock = 16E6

ftot = []
for p in prescaler_values:
  freqs = clock/(2*p*(1+OCRA))
  ftot.append(freqs.data)
  semilogy(OCRA, freqs, '.', label=p)

xlabel('OCRA')
ylabel('PWM frequency')
legend()
grid()
tight_layout()
savefig(r'assets\arduino_digipot_available_frequencies.svg')

ftot = asarray(ftot).ravel()
log_bins = logspace(1,7,(7-1)*10)
plt.hist(ftot, bins=log_bins);
plt.xscale('log')
plt.xlabel('Frequency')
plt.ylabel('Available frequency setpoints')
plt.grid()
savefig(r'assets\arduino_digipot_available_frequencies_hist.svg')
#plt.hlines(5, log_bins.min(), log_bins.max(), linestyle='dashed')

p=1
LO_freq = clock/(2*p*(1+OCRA[-1]))
print(LO_freq)
delta_f = clock/(2*p*(1+OCRA))-LO_freq
semilogy(OCRA, abs(delta_f), '.')
hlines(LO_freq, OCRA[0],OCRA[-1])
