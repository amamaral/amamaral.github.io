A sine-like wave from arduino's PWM

Posted on 26/04/2018

Basic idea: PWM output passes through 2 integrators. Square wave becomes a triangular wave that becomes a parabolic wave with alternating concavities. The total harmonic distortion is relatively small and can be calculated. The only major issue is to calibrate the components feeding the integrators using opamps. A test must be performed…

There are two circuit versions in the folder Dropbox\LabLuzEstruturada\PIBIC\Compartilhamento\Projeto 2 - Modulacao temporal e travamento digital de fase\Pulse-peak2peak_adc. The circuit there shown accepts a square wave input and contains two integrated outputs. The amplitude is set by a single resistor in each stage. Two problems already noticed: the wave oscillate from ground to the 2*amplitude. The behavior would be better if the wave oscillated around the Vpp/2 (There will be problems if the amp tries to push the signal below ground). Also, the + input is referenced from an artificial 2.5V input… in practical terms there must be some noise contributions that will affect the signal balancing.

%pylab inline

x = linspace(0, 2*pi, 5000)
dx = x[1]-x[0]

parabolic_wave = zeros(x.shape)
parabolic_wave[x<=pi] = (4 * x/pi * (1 - x/pi))[x<=pi]
parabolic_wave[x>pi]  = (4 * (x/pi-1)*(x/pi-2))[x>pi]

subplot(211)
plot(x, parabolic_wave, label='parabolic wave')
plot(x, sin(x), label='sin(x)')
grid()

subplot(212)
plot(x, parabolic_wave-sin(x))
ylabel('PW(x)-sin(x)')
grid()

print("THD =", sqrt(sum((parabolic_wave-sin(x))**2 * dx) / sum((sin(x))**2 * dx)))
# THD: 5%

f_clock = 16e6
base_pwm_freq = f_clock/(2*(2**7))
